# APEX UI PRODUCTION DIRECTIVE

Active until explicitly revoked. Overrides all future UI implementation tasks.

---

## SINGLE SOURCE OF TRUTH

The approved Coach Card concept image (Push-Up) is the permanent visual reference for Apex.

- Never redesign it
- Never simplify it
- Never reinterpret it
- Every future UI task must preserve this visual language

---

## FLAGSHIP FIRST

Only the Push-Up card exists until approved.
The Push-Up card becomes the production template for every future exercise.

---

## EVERY FEATURE APPEARS THROUGH THE CARD

Features that must express through the Coach Card:
- Coaching Memory
- Daily Readiness
- Recovery
- Workout Mode
- Pain Detection
- Goal Adaptation
- Progress Tracking

The Coach Card is the primary presentation layer.
The question before every feature: **"How does this appear inside the Coach Card?"**

---

## DAILY READINESS

Before the first workout of each day, show the Daily Readiness check.
The result must directly affect the Coach Card visuals:

| State | Card expression |
|---|---|
| Recovery | calmer visuals |
| Peak readiness | stronger energy |
| Pain | highlighted muscle + warning |
| Deload | recovery styling |

---

## MEMORY

Coaching Memory never displays raw data. Memory changes the Coach Card:
- Different Coach Tip
- Different muscle emphasis
- Different warnings
- Different recovery styling
- Different recommended intensity

User must feel: "The coach remembered me." — not "The app stored data."

---

## COMPLETION CRITERION

A feature is NOT complete until it answers:
**How is this expressed through the Coach Card?**

---

## IMPLEMENTATION ORDER

| Priority | Task |
|---|---|
| 1 | Complete the Push-Up Coach Card |
| 2 | Integrate Daily Readiness |
| 3 | Connect Coaching Memory |
| 4 | Implement adaptive visual states |

Additional exercise cards only after all four are complete.

---

## ENGINEERING RULE

Do not lose the design thread.
Do not switch visual directions.
Do not invent alternative card layouts.
All future UI work extends the approved Coach Card.
